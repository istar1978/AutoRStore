具体的会员划分：
	药店实行会员制，会员暂定分为三个等级：
		初级会员、高级会员和终身会员。当顾客一次性购买药品达到100元即可成为初级会员（视顾客具体意愿而定），
		前台销售职员完成会员卡的激活。每一种药品都有其对应的积分，会员每次购买药品都对积分进行累积，当积分满1000时，
		初级会员自动升为高级会员，当积分满10000时，高级会员自动升为终身会员。药品针对会员级别进行一定的折扣优惠，
		初级会员购买药品实行9.5折标准，高级会员实行9.0折标准，终身会员实行8.5折标准，非会员不进行积分的累加，
		也没有打折优惠。会员卡需药店销售部门职员手工激活，并可以在销售柜台进行密码的修改（初始密码统一设为666666），
		其他具体操作则必须要求操作人员拥有系统管理员权限

一些假设：
	1.同一种药品只有在全部售完之后才能在重新进货
	2.由库管员来判断某种药品是否应该入库（只有满足假设1的药品才可以入库）
	3.同一种药品进货批次不一样，则其药品编号也不一样，因为不同批次药品的价格是可以改变的，不能进行简单的累加
